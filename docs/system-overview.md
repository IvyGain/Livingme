# Living Me システム概要
> 非エンジニアのクライアント向け説明資料
> 作成日: 2026-03-22 / 更新日: 2026-03-24

---

## 📌 このシステムは何をするもの？

**Living Me** は、コミュニティ会員専用のメンバーポータルです。
会員はログインすることで、動画アーカイブ・イベント・日々のジャーナリングテーマなどのコンテンツを楽しめます。
管理者は会員の管理やコンテンツの更新を行います。

---

## 👥 ユーザーの種類（2種類）

| 種類 | 説明 | できること |
|------|------|-----------|
| **通常会員（MEMBER）** | アカウントが有効な会員 | 全コンテンツにアクセス可 |
| **管理者（ADMIN）** | システムの管理権限を持つスタッフ | 会員管理・コンテンツ編集が可能 |

> **停止中の会員**（`isActive=false`）はログイン時にエラーメッセージが表示され、コンテンツにアクセスできません。

---

## 🗺️ 画面の流れ（画面遷移図）

```
【誰でも見られる画面】
  /login（ログイン）
  /join（会員登録・支払い）
  /invite/xxxxx（招待からのパスワード設定）
  /demo（デモ：ログイン不要でお試し閲覧）

        ↓ ログイン後

【正式会員の画面】
  /（ホーム）
  ├─ /archive（動画アーカイブ一覧・詳細）
  ├─ /events（イベント一覧・申込）
  ├─ /journal（ジャーナル）
  │     └─ /journal/new（記入ページ）
  ├─ /forms（申請フォーム）
  └─ /ambassador（アンバサダー専用ページ）

【管理者のみアクセスできる画面】
  /admin（ダッシュボード）
  ├─ /admin/members（会員管理）
  ├─ /admin/content/today（今日のコンテンツ編集）
  ├─ /admin/content/archives（アーカイブ管理）
  ├─ /admin/events（イベント管理）
  ├─ /admin/home-layout（ホーム画面レイアウト設定）
  └─ /admin/settings（外部サービスAPI設定）
```

---

## 📱 各画面の説明

### ログイン関連

#### `/login` ログイン画面
- メールアドレス＋パスワードでログインできます

#### `/invite/[token]` 招待リンクからの登録
- 管理者が送った招待メールのリンクをクリックすると開く画面
- 名前とパスワードを設定して登録完了
- リンクは **72時間有効**・**1回限り**

#### `/join` 会員登録・支払い
- 月額料金を確認して支払い手続きをする画面
- Stripe の安全な決済ページに移動してカード情報を入力

---

### 正式会員の画面

#### `/`（ホーム）
- 今日のエネルギーシェア・ジャーナリングテーマが表示される
- 直近のイベント・アーカイブ・コラムが並ぶトップページ

#### `/archive` 動画アーカイブ
- 過去の動画・セッションを一覧で閲覧
- カテゴリ・タグで絞り込み、キーワード検索も可能

#### `/archive/[id]` アーカイブ詳細
- 動画（YouTube/Vimeo）の視聴
- エネルギーシェア・ジャーナリングテーマ・サマリーを確認

#### `/events` イベント
- 今後のイベント一覧と月間カレンダー表示
- イベントへの申込・キャンセルが可能

#### `/journal` ジャーナル
- 今日のテーマを確認し、日記を書く
- 過去の日記を遡って確認可能

#### `/ambassador` アンバサダー
- 紹介実績・特典の確認
- アンバサダー会員のみアクセス可能

---

### 管理者の画面

#### `/admin` 管理ダッシュボード
- 全会員数・有効会員数・停止中数の統計
- 各管理機能へのクイックアクセス

#### `/admin/members` 会員管理
- 全会員の一覧表示（名前・メール・ステータス・参加日など）
- **「招待メールを送る」ボタン**：メールアドレスを入力すると招待メールが自動送信される
- 会員ごとに「有効/停止切替」「権限変更」が可能

> **招待の流れ**
> 管理者が「招待メールを送る」→ 入力したメールに招待リンク送信
> → 招待受信者がリンクをクリック → パスワードを設定 → ログイン完了

#### `/admin/content/today` 今日のコンテンツ
- 日付ごとに「エネルギーシェア」「ジャーナリングテーマ」「モーニングノート」を設定

#### `/admin/content/archives` アーカイブ管理
- 動画の追加・編集・削除
- タイトル・説明・動画URL・サムネイル・カテゴリ・タグなどを設定

#### `/admin/events` イベント管理
- イベントの追加・編集・削除
- タイトル・日時・場所・Zoom URLなどを設定

#### `/admin/home-layout` ホームレイアウト設定
- ホーム画面に表示するセクションの順番や表示/非表示を設定
- カラーテーマの切り替え

#### `/admin/settings` 外部サービス設定
- Stripe・UnivaPayのAPIキーなど機密情報を安全に保存

---

## 👁️ デモモード（`/demo`）

ログインしなくてもシステムの見た目を確認できるデモページです。
実際のデータではなくサンプルデータが表示されます。
クライアントへの説明・デモに活用できます。

デモで確認できる画面：
- ホーム・アーカイブ・イベント・ジャーナル
- 管理画面（会員管理・コンテンツ編集のデモ）

---

## 🔐 セキュリティの仕組み

| 機能 | 説明 |
|------|------|
| **招待制登録** | 管理者が許可した人のみ登録可能（自己登録不可） |
| **ログイン試行制限** | 5回ログイン失敗すると15分間ブロック（不正アクセス対策） |
| **タイミング攻撃対策** | メールアドレスの存在が外部からわからない設計 |
| **暗号化保存** | APIキーなどの機密情報はデータベースにAES-256-GCMで暗号化して保存 |
| **招待リンクの有効期限** | 72時間で自動失効・1回限り使用 |

---

## 💳 決済の仕組み

対応決済：**Stripe**（クレジットカード決済）、**UnivaPay**（設定準備済み）

| 状態 | 説明 |
|------|------|
| 支払い成功 | 自動的に会員ステータスが「有効」に変更 |
| 支払い失敗 | ステータスが「停止中」に変更、再登録ページへ案内 |
| 解約 | ステータスが「停止中」に変更 |
